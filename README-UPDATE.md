# 로또 데이터 자동 업데이트 가이드

## 현재 상태

현재 구현된 기능은 **다음 회차를 자동으로 계산하여 표시**하는 것입니다. 
실제로 인터넷에서 당첨번호를 가져와 CSV 파일을 업데이트하는 자동화는 **별도로 설정**이 필요합니다.

## 자동 업데이트 방법

### 방법 1: 수동 업데이트 (가장 안정적)

매주 토요일 추첨 후 일요일에 수동으로 업데이트:

1. 동행복권 웹사이트 방문: https://dhlottery.co.kr
2. 최신 회차 당첨번호 확인
3. `data/lotto_1_XXXX.csv` 파일에 다음 형식으로 추가:

```
회차,날짜,번호1,번호2,번호3,번호4,번호5,번호6,보너스
1204,2025.12.27,1,2,3,4,5,6,7
```

### 방법 2: 자동 업데이트 스크립트 (Node.js 필요)

`update-lotto-data.js` 파일이 있지만, 실제 동행복권 API가 변경될 수 있어 **완전 자동화는 어렵습니다**.

#### 설정 방법:

1. Node.js 설치 확인:
```bash
node --version
```

2. 스크립트 실행:
```bash
node update-lotto-data.js
```

3. 자동 실행 설정 (macOS/Linux):
```bash
# crontab 편집
crontab -e

# 매주 일요일 오전 9시에 실행하도록 추가:
0 9 * * 0 cd /path/to/lotto_claude && /usr/bin/node update-lotto-data.js
```

### 방법 3: 웹 스크래핑 (Python, Selenium 등)

Python을 사용한 웹 스크래핑을 통해 자동화할 수 있습니다:

```python
# 예시 (실제 구현 필요)
import requests
from bs4 import BeautifulSoup

# 동행복권 웹사이트에서 최신 회차 정보 가져오기
# ... (구현 필요)
```

## 현재 시스템의 작동 방식

1. **자동 회차 계산**: 현재 날짜 기준으로 다음 회차 번호를 자동으로 계산
2. **추첨일 표시**: 다음 추첨일과 남은 일수를 화면에 표시
3. **예측 대상 표시**: 계산된 다음 회차 번호를 예측 대상으로 표시

## 권장 사항

**가장 안정적인 방법**: 매주 수동으로 업데이트하거나, 
간단한 알림을 설정하여 매주 일요일에 업데이트하도록 리마인더를 받는 것이 좋습니다.

## 문제 해결

- CSV 파일이 업데이트되지 않으면 페이지를 새로고침하세요
- 최신 회차가 표시되지 않으면 브라우저 캐시를 지우고 다시 시도하세요
