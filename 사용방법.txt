🎯 주요 기능

1. 예측 탭
   - 다음 회차 예측 번호 (상위 10개)
   - 각 번호별 출현 가능성 점수
   - 특성별 가중치 시각화

2. 백테스팅 탭
   - 과거 데이터로 모델 검증
   - 회차 범위 설정 가능
   - 적중률 통계 확인

3. 통계 탭
   - 번호별 출현 빈도
   - 구간별 분포
   - 홀짝 비율

4. 설명 탭
   - 엔트로피 가중치법 설명
   - 분석 특성 설명


💡 사용 팁

✅ 분석 회차 조절 (NEW!)
   - "최근 회차 기반" 값 변경 (10~500)
   - 작은 값(예: 30): 최근 트렌드 중시
   - 큰 값(예: 200): 장기 패턴 중시
   - 권장: 50~100 회차

✅ 예측 번호 확인
   - "예측" 탭 → "다음 회차 예측" 버튼 클릭
   - 상위 10개 번호와 점수 확인
   - 특성별 가중치 확인

✅ 조합 생성 (NEW!)
   - 예측 번호에서 자동으로 6개 조합 생성
   - "조합 생성" 버튼 클릭
   - 점수가 높은 조합부터 표시
   - 각 조합 복사 가능 (📋 버튼)

✅ 백테스팅
   - "백테스팅" 탭 이동
   - 시작/종료 회차 설정 (예: 1100~1150)
   - "백테스트 실행" 클릭
   - 성능 지표 및 차트 확인

✅ 통계 확인
   - "통계" 탭에서 전체 번호 빈도 확인
   - 최다/최소 출현 번호 확인


🔧 문제 해결

Q: 데이터가 로드되지 않아요
A: 브라우저 콘솔(F12)을 열어 오류 확인
   - js/lotto-data.js 파일이 있는지 확인
   - 모든 파일이 같은 폴더에 있는지 확인

Q: 차트가 안 보여요
A: 인터넷 연결 확인 (Chart.js CDN 사용)

Q: 느려요
A: 백테스팅 범위를 줄여보세요 (예: 50회차씩)


📊 엔트로피 가중치법이란?

정보 이론의 엔트로피를 이용하여 각 특성의
중요도(가중치)를 객관적으로 결정하는 방법입니다.

- 엔트로피 낮음 → 정보량 많음 → 가중치 높음
- 엔트로피 높음 → 정보량 적음 → 가중치 낮음

8가지 분석 특성:
1. 최근 출현 빈도
2. 미출현 기간
3. 출현 간격 패턴
4. 홀짝 균형도
5. 구간 분포
6. 연속 번호 패턴
7. 보너스 이력
8. 평균 회귀
